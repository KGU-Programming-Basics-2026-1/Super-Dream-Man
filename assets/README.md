# Game resrouce files
Use snake case for all resource files

## Character and animated entities naming convention
```txt
TYPE_NAME_NUMBER.png
TYPE_NAME_CONTAINED-IMAGE_NUM.png
```
- Examples
```txt
player_kiryong_0.png
player_kiryong_1.png
enemies_goomba_1x3.png # 1열에 3개의 상태, 3개 상태가 있을 때
item_coin_2x3.png # 2열에 3개의 상태, 6개 상태
```

## Tile sheet naming convention

```txt
THEME_TYPE.png
```
- Examples
```txt
economics_background.png
economics_block.png or ecomics_breakable.png
economics_unbreakable.png
```

## UI resource naming convention

```txt
PAGE_FEATURE.png
```
- Examples
```txt
start_exit.png
start_new.png
start_continue.png
game_pause.png
game_resume.png
pause_save.png
pause_exit.png
```
